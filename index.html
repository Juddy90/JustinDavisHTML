During the recent pandemic mental health has been a major concern. During 2021, lifeline had seen an increase of 33.1% of daily call volumes (Australian Institute of Health and Welfare 2022), demonstrating a need to focus on mental health. Too often, people have been isolated and haven’t been in contact with others which can result in a feeling of loneliness. Another issue that has occurred with a lack of interaction with other people has been the fact that people haven’t heard what a great job in life they are doing. Parents feel under-appreciated, and workers feel like they are not receiving recognition anymore. This application will work towards reminding them of their worth and will hopefully increase their moral.

Everyday this application will serve the user with a notification showing a positive affirmation or a motivational quote. The user will have the opportunity to give certain details while signing up such as:

  • Male or Female
  • Age
  • Parent

Users under the age of 16 will be asked to submit a parental email address although for users over the age of 12, this will be voluntary. This will be for child safety. These answers will allow the user to be served with different affirmations tailored to their life’s circumstances. For example, if the user is a female parent, the user will be served with affirmations such as “You are doing a great job, mum” or “Your kids may not say it, but they love and appreciate you”, a child of school age may be served with affirmations such as “You are smart and capable of anything you put your mind to”, or an elderly person may be served with affirmations such as “You have people who love you in their lives”. When the application launches the aim is to have roughly 200 different affirmations and motivational quotes available to be given with the aim to increase this to around 500 within 12 months. This would allow for each person to be served a different quote every day for a year instead of being served the same one multiple times in a year. This application will also allow the user to forward any affirmations on to other people they know if they feel they are relevant, and the friend or family member could use some encouragement. Every week this application will request the user take part in a quick mental health quiz of 10 questions which will have the user rate their answers between one and five for questions such as “I feel lonely often?” or “I want this to end?”. This will track any trends and if the trends continue to track lower over a month or if the answers are of high concern, the user will be served with information on lifeline, suicide helplines or emergency services. If the user is a child, the trends and results will be visible to a parent incase there are any concerns that they need to be aware of. While there are similar applications on the market there are no applications that combine the features of positive affirmations, mental health tracking and support services all in one application. The benefits of this application over others is that it will have all of these features built in to one platform while also being a free to use application. In the short term, the minimum viable product will include 200 unique affirmations and quotes and will look to add in features after release such as mental health tracking and support services. This application will be a free application fully funded by advertisements only served if the application is opened by a notification. This ensures there are limited applications, and the user will only get an advertisement if they are interested in the material served.

This application will be designed and released for IOS which is the operating system found on the Apple iPhone. To design this application, the development language that will be used will be Swift. Swift is an open-source programming language that developers use to create applications on iPhones, iPad and MacBooks. Swift works directly with Cocoa and Cocoa Touch which are Apple’s native API and UI frameworks. As there will be data storage and potentially contact details of a parent or guardian in the app, a server will be required to house all of this. The server that will be used will be the Azure SQL database.

The skills required to create and maintain this project will be:

  • Programming skills in Swift
  • An understanding of IOS development
  • An understanding of how Azure SQL database works and how it can be implemented in this scenario

While programming can be hard to learn, there is a wealth of knowledge on Swift, IOS development and Azure SQL Database on social media pages such as YouTube as well as very knowledgeable instruction articles on websites like Udemy which I will be able to utilize to gain the skills required.

The goal for this application is to increase mental health while allows users to connect to each other and share positive thoughts. Users will be able to treat any mental health illnesses early through the tracking system and will have contact details to support systems if they are struggling. This pandemic has been hard of everybody’s mental health and the aim with this application is to increase mental health or at the minimum increase mental health awareness.
